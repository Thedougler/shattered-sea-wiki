# Stat Block References

**For every enemy listed in Challenge Calibration, use this format:**

**Canon stat block** (use whenever a published stat block fits — always prefer this):
> `Merrow` — *Monster Manual p.240*

**Homebrew stub** (only when no published stat block fits the concept):
> `[Custom] Shadow Warden` — See `[campaign-slug]-creature-shadow-warden.md` *(stub — full stat block via creature skill)*

Rules:

- Lean hard on canon. Reskin flavour, keep the math. A pirate enforcer is a Bandit Captain (*MM p.344*). A sea hag cult leader is a Sea Hag (*MM p.179*).
- Only flag homebrew when the mechanical concept genuinely doesn't exist in print.
- Do not generate stat blocks here. The creature skill handles that.
- If the campaign slug isn't known, use `[slug]` as a placeholder.
