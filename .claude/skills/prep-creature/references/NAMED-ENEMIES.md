# Named Unique Enemies → Culturally Grounded Name

**Trigger:** User explicitly names an enemy as unique/named, OR the encounter concept clearly implies a singular named antagonist (e.g., a cult leader, a rival captain, a crime boss). Generic enemies (guards, bandits, wolves) never get this treatment.

**Do not invent named enemies** — if the user hasn't indicated a named foe, the encounter uses generic creature types only.

**When triggered:**

1. Identify the enemy's race and cultural context from the lore provided.
2. Infer the real-world linguistic root using the culturally-grounded-names race mapping (High Elf → Finnish/Welsh, Dwarf → Old Norse, Tiefling → corrupted Latin, etc.).
3. Generate **one name** using phonotactics native to that root. Avoid AI name slop: no Aerin, Theron, Kael, Lyra, Vex, Draven, Zara.
4. Include pronunciation in simple syllable notation: e.g., `KAR-iss`, `HORN-vel`.
5. Output format inside the encounter toy, added as a new row:

| **Named Antagonist** | **Vorrec** *(VOR-ek)* — Half-Orc, Proto-Germanic root, blunt and percussive |
