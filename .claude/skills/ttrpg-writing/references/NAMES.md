# Names Reference

Load this file when choosing a name for an NPC, location, faction, or any proper noun.

---

## Core Rule

Every fantasy culture has a real-world linguistic root. Names should feel like they evolved from that root. Never generate slop defaults: Aerin, Theron, Kael, Lyra, Zara, Mira, Aldric, Seraph, Caelen, Vex, Nyx, Draven, Sylvara, Thorn.

## Step 2: Apply Phonetic Principles

**Finnish:** Open syllables, double vowels (aa, oo, ii), endings -nen, -la, -ri. Stress on first syllable. *Aino, Ilmari, Kullervo.*

**Welsh:** Consonant clusters (ll, rh, ch, dd), w and y as vowels. *Carys, Rhodri, Caradoc.*

**Old Irish / Gaelic:** Lenition (bh = v, mh = w), silent letters. *Aoife, Ciarán, Fergus.*

**Old Norse:** Short, punchy. Elements: -bjorn, -ulf, -heim, -ric, sig-, thor-. *Gunnar, Astrid, Halvard.*

**Scottish Gaelic:** -ach endings, aspirated consonants. *Hamish, Alasdair, Morag.*

**Arabic / Persian:** Tri-consonantal roots, Abd-, Al-, -din prefixes/suffixes. *Farouk, Tariq, Zainab.*

**Nahuatl:** tl, tz, x (= sh), long compounds. *Xochitl, Itzcoatl, Coatl.*

**Latin / Corrupted Latin:** Case endings; tieflings corrupt them. *Valeria, Cassius, Morthos, Damaia.*

**Japanese:** Mora structure, endings -ko, -ro, -mi, -ta, -shi. *Takeshi, Haruto, Mizuki.*

**Slavic:** -av, -ov, -mir, -slav endings. *Miroslav, Radovan, Bozena.*

---

## Step 3: Anti-Patterns Checklist

Before using any name, check:
- Ends in a decorative apostrophe? (`Ae'rin`) — cut it.
- Sounds like a generic fantasy video game protagonist? — replace it.
- Real English word with a letter swapped? (`Ryvyn` for Raven) — cut it.
- Unpronounceable without a guide? — simplify.

---

## Place Names

Place names follow the same root. Examples:
- Finnish: -järvi (lake), -mäki (hill)
- Welsh: Aber- (river mouth), Llan- (enclosure)
- Norse: -heim, -vik, -fjord, -dal
- Arabic: Wadi- (valley), Bab- (gate), Dar- (house of)
