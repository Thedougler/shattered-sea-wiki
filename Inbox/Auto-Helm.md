# Auto-Helm

## Description

A clockwork mechanism fitted to the ship's tiller or wheel.

## Properties

- **[HB]** Price: **1,400 gp**
- Holds a set course without a crew member at the helm for up to 8 hours.
- Does not react to other vessels, obstacles, or changing weather.
- Requires a course to be set manually before engaging.
- The Navigator role must still be filled for open-water navigation — the Auto-Helm holds a heading, it does not plot one.
- Disengages automatically in winds above Force 6.
