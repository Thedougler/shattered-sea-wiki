# Cartographer's Table

## Description

A fixed chart table that records route data in real time for navigation and trade.

## Properties

- **[HB]** Price: **8,000 gp**
- Automatically maps the ship's route in real time.
- Produced charts are saleable for 50–200 gp each to merchants.
- Does not function over the Maw below 200-foot depth.
