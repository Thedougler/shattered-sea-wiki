# Pneumatic Grapple

## Description

A wrist-mounted device that fires a grappling hook up to 60 feet.

## Properties

- **[HB]** Price: **600 gp**
- Fires a grappling hook up to 60 feet as a bonus action.
- Line retracts mechanically, pulling the user toward a fixed point (STR 18 for the pull).
- Three shots before requiring 10 minutes to repressurize from the included hand pump.
- Hook and 60 feet of braided steel line are standard components — replacements cost 15gp.

## Current Status

Available from [[Catarina-DaVirelli]] by referral.
