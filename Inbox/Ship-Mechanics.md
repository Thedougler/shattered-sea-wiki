# Ship Mechanics

Rules hub for the party ship. The ship functions as vehicle, home base, and mobile bastion.

Ship combat and chases use the **2024 DMG vehicle rules** as written.

## Quick Summary

**Ship tier** (1–4) sets everything: cargo capacity, guns, crew requirements, upkeep cost, and which bastion facilities fit aboard. See [[Ship-Stats]].

**Weekly upkeep** runs 20–40 gp (Tier 1), 80–145 gp (Tier 2), or 270–430 gp (Tier 3). See [[Ship-Operations]].

**The ship is a shared bastion.** Facility slots pool across all PCs. Bastion Turns resolve weekly. See [[Ship-Bastion]].

**Magic enhancements** are professional tools, not wonders. Common items are available in Tidefall; rare items go through [[Catarina-DaVirelli|Catarina Da'Virelli]]. See [[Ship-Upgrades]].
