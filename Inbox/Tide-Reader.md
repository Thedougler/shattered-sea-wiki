# Tide Reader

## Description

A mounted hydrographic instrument for current and tidal forecasting.

## Properties

- **[HB]** Price: **600 gp**
- Reads current speed and direction to 20-foot depth.
- Predicts tides for known ports within 200 miles.
- Grants advantage on navigation checks in reef-heavy waters.
