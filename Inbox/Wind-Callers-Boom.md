# Wind Caller's Boom

## Description

A mainmast installation that can force favorable wind for a limited duration.

## Properties

- **[HB]** Price: **1,200 gp**
- Fitted to the mainmast.
- Once per day: favorable wind for 4 hours; +30 miles/day regardless of conditions.
- Does not function in a storm or the Doldrums.
